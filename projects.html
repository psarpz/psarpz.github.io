Rock,Paper, Scissors Game
  • Rock, Paper, Scissors is a popular game played in many households or places.
  • It is often used as a decision-making tool between two people.
  • The game is based on the idea of random chance and is typically played by making one of three possible hand shapes:
  • Rock, Paper, or Scissors.
  • Here's how the game works:
  • Rock beats Scissors (Rock crushes Scissors).
  • Scissors beat Paper (Scissors cut Paper).
  • Paper beats Rock (Paper covers Rock).
  • Eg, if one person chooses Rock and the other chooses Scissors, the player with Rock wins because Rock beats Scissors.

Guess the right number game.

Guess the Right Number game is a common and straightforward number-guessing game that is often used for entertainment purposes.
The game typically follows these basic rules:
The game selects a random number within a specified range (e.g., between 1 and 100).With my version, the player's objective is to guess the selected number within three(3) attempts. After each guess, the game provides feedback to the player, indicating whether the guessed number is too high, too low, or correct.The player continues guessing until they either correctly guess the number or exhaust their allowed number of attempts.
The game then announces whether the player has won (guessed the number) or lost (failed to guess the number within the allowed attempts).


Many password resets at a go vis powershell.
  • Sets the target domain and prompts the user to enter their credentials.
  • Defines a function Get-RandomCharacters to generate random characters for passwords.
  • Defines a function Scramble-Password to scramble a password's characters.
  • Defines the Update-Password function to reset a user's password, including generating a new password and updating user properties.
  • Reads a list of usernames from a CSV file and stores it in the $Usrs variable.
  • Retrieves a list of domains in the Active Directory forest and stores it in the $srvs variable.
  • Checks if the CSV file containing usernames exists. If not, it displays an error message.
  • Loops through each domain and each username, validates the existence of the user, and attempts to reset their password using the Update-Password function.
  • Logs successful password resets to a CSV file named "RandomBulkPWGood.csv" and logs errors to "RandomBulkPWError.csv."
  • Calculates the total number of password resets performed and displays it.
  • Indicates the completion of the script and the location of the log file.
